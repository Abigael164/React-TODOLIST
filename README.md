#  React To-Do List App

A simple and interactive **To-Do List application** built with **React** that allows users to **add, edit, and delete tasks** easily.  
Perfect for learning and practicing React state management and component design.

---

##  Features

- Add new tasks  
- Update existing tasks  
- Delete tasks  
- Data persistence using local storage  
- Clean and responsive User Interface

---

## Technologies Used

- **React** (useState, useEffect)
- **JavaScript (ES6+)**
- **HTML5 & CSS3**
- **Vite**
